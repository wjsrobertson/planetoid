Planetoid

Planetoids is an open-source game written in Scala, Java and JavaScript

This game was written as an experiment on using WebSockets

The source code is available on the planetoids GitHub page

The images used in the game are used under Creative Commons license - attribution follows:

  • Background star image by n4pgamer
  • Spaceships & missiles by Kenney.nl
  • Explosions by Soluna Software

Sounds created using an Access Virus TI Snow synth

Planetoid is actually a real word, synonymous with asteroid
